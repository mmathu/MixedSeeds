About


MixedSeeds works similar to an RSS feed, in which it displays headlines from various websites of the user's interest. The headlines are retrieved based on keywords that have been predeterimed by the user.


The website allows the user to only visit one website to retrieve all the articles that they are interested in.


MixedSeeds has been built and developed using HTML, CSS, and PHP5 by Myoran.
